# Проект фильмотеки на ReactJS
____
API: [OMDBAPI.COM](https://www.omdbapi.com)
____
## Установка всех пакетов
>npm i
## Запуск проекта
>npm start
## Сформировать для разработки
>npm run build:dev
## Сформировать для продукта
>npm run build:prod
___
### Возможности:
- Поиск по фильмам
- Фильтрация по фильмам
